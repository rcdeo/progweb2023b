Parâmetros:

  • ...

  • loop
  • muted

  • ...
  • controls
  • ...
  • video: width and/or height
  • video: poster (capa)
  • video: track (legendas)

Observe bem os formatos de arquivo suportados. Procurem colocar MP3 que é um formato mais genérico para áudios e MP4 para os vídeos. Além dos formatos populares, está em constante crescimento a adoção do formato AV1 para vídeos.

mdn web docs

Nota: O atributo autoplay tem precedência sobre o preload, pois se é necessário executar o áudio automaticamente, o navegador obviamente o baixará. Definindo ambos autoplay e preload é permitido pela especificação.
